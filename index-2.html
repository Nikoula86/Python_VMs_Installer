Python_VMs_Installer

  1. open environment.yml with any text editor and change the name of the environment to whatever you like (currently "nicola")

  2. install environment:

  3. open command prompt in any location on the VM

  4. type "conda env create -f environment.yml" and press enter

  5. install opencl:

  6. open a command prompt in the location of the pyopencl file

  7. type "pip install " and press enter

  8. install pyclesperanto for fast image processing:

  9. in the same command prompt, type "pip install pyclesperant_prototype"
